# ML Algorithms
## To Contribute
- fork this repository
- git checkout -b path_branch_name
- git add .
- git commit -m "commit message"
- git push origin main
- come to github repo, you will get notification to compare and pull request 
 
## To Execute
- python3 -m venv env_name
- Windows:
 env_name/Scripts/activate
- Mac/Linux:
 source env_name/bin/activate
- pip install -r requirements.txt
- execute any ipynb in VS code by changing the kernel to newly created environment
- note: you can also use kaagle or colab for execution (no need to follow above steps)
